# Purli-s-repository
Slack Bot Delilah6312 - a Slack Bot that replies to commands 24/7. It is not too developed yet, but the creator purlimurlikas does plan to add more commands in the future. 
Delilah6312 has 4 commands: /delilah6312-pong - Check bot latency, /delilah6312-catfact - Get a cat fact, /delilah6312-joke - Get a joke, /delilah6312-dadjoke - Get a dad joke, /delilah6312-foodpicture - Get a food picture, and /delilah6312-help - says available commands.

This is the channel in Slack for Slack Bot Delilah6312: https://hackclub.enterprise.slack.com/archives/C0BF89U3RRP 
To use the slack bot, type in any of the available commands in a Hack Club slack channel, click 'send' and the bot will do the rest of the work.

When you send a command, the bot will with what the command has been programmed for it to respond with, for example: /delilah6312-joke will make the bot fetch a joke from  "https://official-joke-api.appspot.com/random_joke"
